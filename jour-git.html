COMMANDES DESCRIPTIONS
Git config Définition de l'email que vous voulez associer a toutes vos opérations de commit
Git config Active la colorisation de la sortie en ligne de commande
Git init Crée un dépôt local à partir du nom spécifié
Git clone Télécharge un projet et tout son historique de version
Git status Liste tous les nouveaux fichiers et les fichiers modifiés a commiter
Git diff Montre les modifications de fichier pas encore indexées
Git add Ajoute un instantané du fichier, en préparation pour le suivi de version
Git diff Montre les différences de fichier entre la version indexée et la dernière version.
Git reset Enleve le fichier de l'index, en conservant son contenu
Git commit Enregistre des instantanés de fichiers de façon permanente dans l'historique des versions.
Git branch Liste l'ensemble des branches locales dans le dépôt courant
Git branch Création d'une nouvelle branche
Git checkout Bascule sur la branche spécifiée et mise à jour du répertoire de travail
Git merge Combine dans la branche courante l'historique de la branche spécifiée
Git branch Supprime la branche spécifiée
Git rm Supprime le fichier du répertoire de travail et met à jour l'index
Git mv Renomme le fichier et prépare le changement pour un commit
Git ls Liste tous les fichiers exclu du suivi de version dans ce projet
Git stash Enregistre de manière temporairement tous les fichiers sous suivi de version qui ont été modifiés
Git stash pop Applique une remise et la supprime immédiatement
Git stash list Liste toutes les remises
Git stash drop Supprime la remise la plus récente
Git log Montre l'historique des versions pour la branche courante
Git diff Montre les différences de contenu entre 2 branches
Git show Montre les modifications de métadonnées et de contenu inclues dans le commit spécifié
Git reset Supprime tout l'historique et les modifications effectuées après le commit spécifié
Git fetch Récupère tout l'historique du dépôt nommé
Git merge Fusionne la branche du dépôt dans la branche locale courante
Git push Envoie tous les commits de la branche locale vers GitHub
Git pull Récupère tout l'historique du dépôt nommé et incorpore les modifications
